# Sistema de Información Web

Este proyecto corresponde al desarrollo de un sistema de información web realizado para la universidad.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Visual Studio Code

## Cómo ejecutar el sistema

Si se descarga el proyecto:

1. Abrir el archivo **index.html** en el navegador.
2. El sistema iniciará en la página de inicio de sesión.
3. Primero debe registrarse un usuario.
4. Luego podrá iniciar sesión con las credenciales registradas.

## Funcionalidades

- Registro de usuario
- Inicio de sesión
- Validación de credenciales
- Página principal del sistema

## Enlace del sistema en línea

El sistema también se encuentra publicado mediante GitHub Pages.
